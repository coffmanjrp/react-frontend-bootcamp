## How to start

1. Clone this repository.

```
git clone git@github.com:coffmanjrp/react-bs5-frontend-bootcamp.git
```

Or download the zip file.

2. Run following command to install the dependencies.

```
npm install
# or
yarn
```

3. Set your [mapbox](https://www.mapbox.com/) key to the environment variable.

example:

```
# .env.local file

REACT_APP_MAPBOX_TOKEN=YOUR_MAPBOX_TOKEN
```

4. And you can start localhost by

```
npm start
# or
yarn start
```

Enjoy!

## Stack

- [React](https://reactjs.org/)
- [mapbox](https://www.mapbox.com/)
- [Bootstrap](https://getbootstrap.com/)
- [React Bootstrap Icons](https://github.com/ismamz/react-bootstrap-icons)
